1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/kr1pt0n/phpyshell.git
   cd phpyshell

2. **Mover al servidor web:**
   ```bash
   sudo cp shell.php /var/www/html/
   
3. **Acceder:**
   ```bash
   Localiza la webshell en http://dominio.com/shell.php
